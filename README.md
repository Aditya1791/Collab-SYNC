# Collab-SYNC Pro

Collab-SYNC Pro is a real-time collaborative Kanban board (similar to Trello or Jira) built for teams to manage tasks, collaborate in real-time, and track project activities.

## Features

- **User Accounts & Workspaces:** Sign up, log in, manage profile colors/avatars, create workspaces, and invite team members.
- **Kanban Boards:** Create custom columns (e.g., *To Do*, *In Progress*, *Done*) and organize tasks in them.
- **Drag-and-Drop Task Management:** Move task cards across different columns.
- **Real-time Collaboration:** Multiple users can view the same board simultaneously, with live presence avatars and live drag-and-drop sync.
- **Task Customization:** Manage task priorities, assignees, due dates, checklists, and descriptions.
- **Board Activity Feed:** A live log displaying actions taken on the board.

## Run Locally

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
